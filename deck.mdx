# About animations in frontend

---

## Main ways

- JavaScript(setTimeout, setIterval, requestAnimationFrame)
- [CSS3](https://developer.mozilla.org/zh-CN/docs/Web/CSS/CSS3)
- HTML5(canvas, svg, webgl)

```notes
JavaScript：通过定时器（setTimeout 和 setIterval）来间隔来改变元素样式，或者使用requestAnimationFrame；
CSS3：transition 和 animation；
HTML5：使用HTML5提供的绘图方式（canvas、svg、webgl）；
```

## Point!

- [`@keyframes`](https://developer.mozilla.org/en-US/docs/Web/CSS/@keyframes)
  通过关键帧定义来控制动画
- [`transform`](https://developer.mozilla.org/zh-CN/docs/Web/CSS/transform)
  属性允许你旋转，缩放，倾斜或平移给定元素。这是通过修改 CSS 视觉格式化模型的坐标空间来实现的。
- [`animation`](https://developer.mozilla.org/en-US/docs/Web/CSS/animation)
  是如下属性的一个简写属性形式: animation-name, animation-duration, animation-timing-function, animation-delay, animation-iteration-count, animation-direction 和 animation-fill-mode.
